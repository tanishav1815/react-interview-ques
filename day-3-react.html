ques:-- Difference between Axios and fetch

        ans :-- Axios has url in request object. Fetch has no url in request object. Axios is a stand-alone third party package that can be easily installed. Fetch is built into most modern browsers; no installation is required as such.

        


        ques:-- How do you pass data from parent to child?

        ans:--  Data can be passed from the parent to the child component via props. Any properties of the parent component, be it variables or functions alike, can be passed to the child component as props.
